# 一、[GIT](https://learngitbranching.js.org/)

## 常用命令

- 克隆远程库	`git clone 地址`
- 添加到暂存    `git add .`

<!--more-->

- 提交暂存区的文件到本地仓库    `git commit -m 'test'`
- 拉取    `git pull origin <name>`
- 推送    `git push origin <name>`
- 查看仓库当前的状态    `git status`
- 查看远程库信息    `git remote`
- 查看当前分支    `git branch` 
- 切换分支    `git checkout <name>`
- 合并分支到当前分支    `git merge <name>`
